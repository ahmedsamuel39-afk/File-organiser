# File-organiser
Project Overview

A Python tool that automatically organises coding project folders into structured categories such as:

projects
unfinished
learning

Custom categories can be easily added.

Purpose

Built to solve the problem of disorganised coding directories when working on multiple projects, practice problems, and experiments.

The tool provides a simple way to structure and manage development workspaces efficiently.

Features
Scans a root coding directory for project folders
Filters relevant directories while ignoring system and category folders
Allows interactive classification of projects
Automatically moves folders into structured categories
Includes safeguards to prevent duplicate or invalid operations

Key Concepts Demonstrated
File system automation using Python
Rule-based logic and conditional processing
Structured handling of real-world data (directories and files)
User interaction and input-driven workflows

Use Case

Useful for developers, students, and self-taught programmers managing multiple coding projects who want an automated way to organise their workspace.
