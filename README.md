# File-organiser
A Python tool that automatically organizes coding projects into categories such as (you can create your own custom categories):

- practice
- projects
- unfinished
- learning

This was built to solve the problem of messy coding folders
when learning programming.
Features

• scans a coding directory
• classifies projects interactively
• automatically moves folders
• prevents duplicate sorting
Useful for developers, students, and self-taught programmers who manage multiple coding projects, practice problems, and experiments, and want an automated way to organise their workspace.
